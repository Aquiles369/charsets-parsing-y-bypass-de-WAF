Security Policy
Supported Versions

Este proyecto se encuentra en investigación activa.
La versión principal disponible en la rama main es la única mantenida y actualizada.

No se garantiza soporte para forks o versiones modificadas por terceros.


Responsible Use

El contenido debe utilizarse exclusivamente en:

Laboratorios controlados

Programas de bug bounty autorizados

Auditorías con permiso explícito

El autor no se responsabiliza por uso indebido fuera de contextos legales y éticos.
